# Changelog

## Glass UI v1
- Полный редизайн интерфейса в стиле «стекло» (Apple-like glass UI)
- Drag & drop зона загрузки изображения
- Улучшенный прогресс OCR + тост‑уведомления
- Переключатель OCR: локально (Tesseract.js) / Cloud endpoint
- Препроцессинг изображения (Otsu + контраст + автопереворот) для повышения качества OCR
- Фильтр по E‑кодам в таблице
- Новый формат истории с действиями «Открыть» и «Копировать текст»
- Обновлённая карточка‑экспорт (PNG) в новом стиле
- Обновлён service worker и manifest
